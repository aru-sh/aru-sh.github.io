Binary Search

In Binary Sort, we have to first sort the array or sorted array is already given.
Lets suppose the element to be found is num. The next step is to define element mid which lies in the mid of the array given.
Then we compare if the num==arr[mid]. If it is the case then we return arr[mid].
Else if num<arr[mid] then we recursively call the first half of the array(excluding mid).
Else we call binary search on the second half of the array.
The C++ code for binary code is given below.


#include<iostream>
using namespace std;

int binarysearch(int arr[],int num,int start,int end){
    int mid=end+start/2;
    
    if(arr[mid]==num){
         return mid;
    }
    else if(num<arr[mid])
    {
        return binarysearch(arr,num,0,mid-1); 
    }
    else if(num>arr[mid])
    {   
        return binarysearch(arr,num,mid+1,end);  
    }

    }
